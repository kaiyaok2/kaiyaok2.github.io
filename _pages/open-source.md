- **The International Dataset of Flaky Tests (IDoFT)**

  [My commits](https://github.com/TestingResearchIllinois/idoft/commits?author=kaiyaok2) 路

- **NIOInspector**  
  Non-anonymized implementation of *NIODebugger* - a tool to fix non-idempotent flaky tests, released to Maven Central.  
  [NIOInspector v1.1.0](https://mvnrepository.com/artifact/edu.illinois/NIOInspector/1.1.0)

- **NonDex**  
  Contributions to tooling for detecting and debugging false assumptions on under-determined Java APIs.  
  [NonDex Maven Plugin v2.2.1](https://mvnrepository.com/artifact/edu.illinois/nondex-maven-plugin/2.2.1) 路
  [NonDex Gradle Plugin v2.2.1](https://plugins.gradle.org/plugin/edu.illinois.nondex)

- **pytest-ranking**  
  Python testing utility for test case prioritization.  
  [pytest-ranking v0.3.4](https://pypi.org/project/pytest-ranking/)

- **Large-Scale Flaky Test Repair**  
  Fixed **300+ flaky tests** across multiple large-scale open-source projects.
